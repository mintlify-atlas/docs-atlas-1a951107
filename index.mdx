## What is DeenPAL?

DeenPAL is a Retrieval-Augmented Generation (RAG) based chatbot designed to provide accurate and personalized Islamic information from trusted Hadith sources. Built on modern AI technologies, DeenPAL combines semantic search with large language models to deliver contextually relevant answers grounded in authentic Islamic texts from Sahih Bukhari and Sahih Muslim.

## How It Works

DeenPAL processes Hadith collections through a sophisticated pipeline:

1. **Data Loading** - Extracts and processes Hadith PDFs with metadata preservation
2. **Semantic Splitting** - Chunks text based on Hadith structure (chapters and book numbers)
3. **Vector Embedding** - Generates embeddings using sentence-transformers
4. **Retrieval** - Finds relevant Hadiths using MMR-based search
5. **Generation** - LLM synthesizes answers with proper citations

<Note>
DeenPAL uses the `all-MiniLM-L6-v2` model for embeddings and DeepSeek LLM via OpenRouter API for response generation.
</Note>
